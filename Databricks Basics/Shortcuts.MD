## Databricks has two modes:

**Edit mode** = typing inside a cell
**Command mode** = controlling the whole cell
Press **Esc** to go to command mode, and **Enter** to go back to edit mode. ([Databricks Docs][1])



### Must-know shortcuts

| Action                               |             Windows shortcut |             Mac shortcut |
| ------------------------------------ | ---------------------------: | -----------------------: |
| Run current cell                     |             **Ctrl + Enter** |          **Cmd + Enter** |
| Run cell and go next                 |            **Shift + Enter** |        **Shift + Enter** |
| Run cell and create new below        |              **Alt + Enter** |       **Option + Enter** |
| Run selected text / active statement |     **Ctrl + Shift + Enter** | **Ctrl + Shift + Enter** |
| Format code                          |         **Ctrl + Shift + F** |      **Cmd + Shift + F** |
| Comment / uncomment line             |                 **Ctrl + /** |              **Cmd + /** |
| Autocomplete                         |             **Ctrl + Space** |         **Ctrl + Space** |
| Open command palette                 |         **Ctrl + Shift + P** |      **Cmd + Shift + P** |
| Find                                 |                 **Ctrl + F** |              **Cmd + F** |
| Insert cell above                    |        **A** in command mode |                    **A** |
| Insert cell below                    |        **B** in command mode |                    **B** |
| Delete cell                          |     **D, D** in command mode |                 **D, D** |
| Copy cell                            | **Ctrl + C** in command mode |              **Cmd + C** |
| Cut cell                             | **Ctrl + X** in command mode |              **Cmd + X** |
| Paste cell                           | **Ctrl + V** in command mode |              **Cmd + V** |
| Toggle output                        |        **O** in command mode |                    **O** |
| Show shortcut menu                   |        **H** in command mode |                    **H** |
| Toggle line numbers                  |        **L** in command mode |                    **L** |
| Clear state and outputs              |                     **0, 0** |                 **0, 0** |
